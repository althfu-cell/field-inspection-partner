# 현장점검 파트너 · 시흥시

시흥시 현장점검 담당자를 위한 모바일 최적화 웹 앱입니다.  
점검지 주소만 입력하면 카카오맵·네이버지도 연동부터 점검 체크리스트까지 바로 시작할 수 있습니다.

---

## 주요 기능

- **네비 연동** — 점검지 주소 입력 시 카카오맵·네이버지도 바로 연결
- **점검 체크리스트** — 불법광고물·공중화장실·공원시설·도로시설 등 유형별 항목 제공, 수정·추가·삭제 가능
- **결과 기록** — 점검지별 완료 / 이상 발견 / 재방문 선택
- **완료 셀레브레이션** — 모든 점검지 완료 시 컨페티 애니메이션과 함께 결과 요약 표시
- **링크 공유** — 점검표를 URL로 공유, QR코드 배포 가능
- **진행 상황 저장** — localStorage를 통해 페이지를 닫아도 진행 상황 유지

---

## 사용 방법

1. 점검 유형·날짜·점검지 주소를 입력합니다
2. **점검표 만들기** 버튼을 누릅니다
3. 현장에서 카카오맵 / 네이버지도로 이동하며 체크리스트를 완료합니다
4. 점검지마다 결과 버튼(완료·이상 발견·재방문)을 선택합니다
5. 모든 점검지 완료 시 셀레브레이션 화면이 자동으로 나타납니다

---

## 기술 스택

- 단일 HTML 파일 (Vanilla JS / CSS, 외부 의존성 없음)
- OpenStreetMap Nominatim — 주소 자동완성
- LocalStorage — 진행 상황 저장
